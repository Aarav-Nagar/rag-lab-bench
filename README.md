# RAG Lab Bench

RAG Lab Bench compares retrieval-augmented generation strategies with clear fixtures and repeatable metrics.

The goal is to help builders see how chunk size, overlap, retrieval count, and citations change answer quality.

## Planned Features

- Deterministic document fixtures.
- Chunking strategy comparison.
- Retrieval recall and citation hit-rate metrics.
- Visual report for strategy tradeoffs.

## Quick Start

```powershell
python -m rag_lab_bench --demo
```
